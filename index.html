Cisco CCNA ITN, SRWE, ENSA

Digitalent Kominfo

Cisco CCNA (Cisco Certified Network Associate) is a certification for network professionals that demonstrates the ability to design, install, configure, operate, and maintain computer networks.

CCNA ITN (Introduction to Networks) is one of the course modules for CCNA certification that covers basic concepts of computer networks, including protocols, architecture, topology, the OSI (Open Systems Interconnection) model, and network configuration.

CCNA SRWE (Switching, Routing, and Wireless Essentials) is one of the course modules for CCNA certification that covers switching and routing concepts, including VLAN, Spanning Tree Protocol (STP), OSPF (Open Shortest Path First), and wireless network configuration.

CCNA ENSA (Enterprise Networking, Security, and Automation) is one of the course modules for CCNA certification that covers enterprise network concepts, network security, and network automation, including firewall configuration, VPN (Virtual Private Network), and network automation using Python.
